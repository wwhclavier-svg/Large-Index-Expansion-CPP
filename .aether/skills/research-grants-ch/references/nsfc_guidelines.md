# 国家自然科学基金 (NSFC) 申请指南

## 概述

国家自然科学基金是我国支持基础研究的主渠道，由国家自然科学基金委员会管理。本指南涵盖青年科学基金项目和面上项目两种主要项目类型。

## 项目类型与特点

### 青年科学基金项目

**定位**：支持青年科学技术人员在国家自然科学基金资助范围内自主选题，开展基础研究工作。

**申请条件**：
- 男性未满35周岁，女性未满40周岁（按申请当年1月1日计算）
- 具有博士学位
- 具有从事基础研究的经历
- 未获得过青年基金项目资助

**资助强度**：30-60万元/项
**资助期限**：3年
**申请时间**：每年3月集中接收期

**申请要点**：
- 强调创新性和可行性
- 研究内容不宜过大，聚焦关键科学问题
- 申请人需展示独立科研能力
- 预期成果应具体可衡量

### 面上项目

**定位**：支持科学技术人员在国家自然科学基金资助范围内自主选题，开展创新性的科学研究。

**申请条件**：
- 具有高级专业技术职务（职称）或博士学位
- 具有从事基础研究的经历
- 无年龄限制

**资助强度**：60-100万元/项（依学科不同）
**资助期限**：4年
**申请时间**：每年3月集中接收期

**申请要点**：
- 研究内容要有一定深度和广度
- 突出研究的创新性和科学意义
- 需要有较好的研究基础和前期积累
- 团队配置合理

## 申请书结构（2026年版）

申请书正文原则上不超过30页，鼓励简洁表达。请勿删除或改动下述提纲标题及括号中的文字。

### （一）立项依据

**撰写要求**：为什么要开展此项研究，研究的科学技术价值如何

**包含内容**：
1. 研究背景与意义
2. 国内外研究现状及分析
3. 研究的科学技术价值

**撰写要点**：

**研究背景与意义**：
- 从大背景切入，逐步聚焦到具体问题
- 阐述研究的科学价值和应用前景
- 说明为什么要研究这个问题
- 强调研究的必要性和紧迫性

**国内外研究现状及分析**：
- 系统梳理领域发展脉络
- 按主题或方法分类综述
- 指出现有研究的不足和空白
- 明确本研究的切入点和创新点
- 文献评价而非简单罗列

**研究的科学技术价值**：
- 科学价值：理论贡献、学科推动
- 技术价值：方法创新、应用前景

### （二）研究内容

**撰写要求**：提纲不做限制，请按照研究工作的自身逻辑撰写。应提炼出特色与创新点、年度研究计划

**包含内容**（可按自身逻辑自由组织）：

**研究目标**：
- 明确、具体、可检验
- 层次分明，主次有序
- 与研究内容呼应

**研究内容**：
- 围绕目标展开，逻辑清晰
- 各部分相互关联，形成整体
- 工作量适中，切实可行

**拟解决的关键科学问题**：
- 提炼核心问题，通常2-3个
- 问题要有深度，非技术问题
- 与研究目标紧密对应

**研究方法与技术路线**：
- 与研究内容一一对应
- 技术路线清晰可行
- 方法先进、适用

**本项目的特色与创新之处**：
- 理论创新：新概念、新模型、新规律
- 方法创新：新方法、新技术、新算法
- 应用创新：新应用、新领域

**年度研究计划**：
- 时间节点明确
- 任务分配合理
- 与预期成果对应

**预期研究成果**：
- 论文发表计划（数量、质量）
- 专利申请计划
- 人才培养计划

### （三）研究基础

#### 1. 研究基础与可行性分析

**与本项目相关的研究工作积累**：
- 前期研究成果
- 相关论文发表情况
- 预实验数据

**已取得的研究工作成绩**：
- 已发表的论文
- 已申请/授权专利
- 相关项目经历

**研究风险的应对措施**：
- 潜在风险分析
- 应对策略
- 备选方案

#### 2. 工作条件

**已具备的实验条件**：
- 实验平台和设备
- 技术平台
- 数据资源

**尚缺少的实验条件和拟解决的途径**：
- 缺少的设备或资源
- 获取途径和计划

**利用国家实验室、全国重点实验室和部门重点实验室等研究基地的计划与落实情况**：
- 可利用的平台资源
- 合作计划
- 落实情况

### 参考文献

**要求**：
- 青年基金：约30篇
- 面上项目：约40-50篇
- 近5年文献占比不低于60%
- 中英文文献合理搭配
- 格式统一规范

## 评审要点

### 通讯评审

评审专家从以下方面评价：

1. **科学价值与创新性** (30-40分)
   - 研究的科学意义
   - 创新性程度
   - 对学科发展的推动作用

2. **研究内容的合理性** (20-30分)
   - 研究目标是否明确
   - 研究内容是否充实
   - 各部分逻辑是否清晰

3. **研究方法的可行性** (20-30分)
   - 研究方法是否科学
   - 技术路线是否合理
   - 方案是否可行

4. **研究基础与条件** (10-20分)
   - 前期研究积累
   - 研究条件和平台
   - 团队实力

### 会议评审

在通讯评审基础上，会议评审更关注：

- 研究方向的前沿性
- 申请人的学术潜力
- 与基金委资助政策的契合度
- 与同类项目的比较

### 不同项目类型侧重点

| 项目类型 | 评审侧重 |
|----------|----------|
| 青年基金 | 创新潜力、独立研究能力 |
| 面上项目 | 学术积累、研究深度 |

## 常见问题

### 申请阶段

1. **选题不当**：问题太大或太小，缺乏科学价值
2. **创新不足**：低水平重复，缺乏原创性
3. **方案模糊**：技术路线不清晰，方法描述笼统
4. **基础薄弱**：前期积累不够，缺乏支撑

### 格式问题

1. **篇幅过长**：超过30页
2. **参考文献格式不规范**：与基金委要求不符
3. **图表质量差**：分辨率低，标注不清
4. **附件材料缺失**：缺少必要的证明材料

## 重要时间节点

| 时间 | 事项 |
|------|------|
| 每年1月 | 发布申请指南 |
| 3月20日左右 | 集中接收期截止 |
| 4-5月 | 形式审查 |
| 5-7月 | 通讯评审 |
| 8-9月 | 会议评审 |
| 9-10月 | 公示资助结果 |
| 次年1月 | 项目启动 |

## 注意事项

1. **提前准备**：建议提前3-6个月开始准备申请书
2. **多轮修改**：申请书需反复打磨，至少修改5-10稿
3. **同行评议**：请同行专家提意见，特别是不同领域的专家
4. **关注指南**：仔细阅读年度申请指南和限项规定
5. **系统熟悉**：提前熟悉ISIS系统操作，避免最后时刻技术问题
